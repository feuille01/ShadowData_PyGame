### Shadow Data — это атмосферная 2D-игра в жанре стелс с элементами хоррора и головоломок. Игрок должен проникать в секретные офисы, избегать патрулирующих врагов, собирать важные документы и выполнять задания, чтобы раскрыть все тайны корпораций.

![image](https://github.com/user-attachments/assets/8542864a-40ae-4c2e-bb3f-3cb7f4fc0d9c)



### Описание игры
В Shadow Data вы играете за хакера, который должен проникнуть в секретные офисы корпораций. Ваша задача — собрать важные документы, избегая патрулирующих охранников. Используйте укрытия и свои навыки скрытности, взламывайте компьютеры и собиарйте важные документы, чтобы выполнить миссию и выбраться непойманным.

Игра сочетает элементы стелса, головоломок и хоррора, создавая напряжённую атмосферу. Вдохновением послужили такие игры, как Hotline Miami, Mark of the Ninja и другие инди-стелсы.

### **Геймплей**  
- **Управление:**  
  - `WASD`: Движение персонажа.  
  - Прятаться за объектами (ящики, стены).  
  - Взаимодействовать с компьютерами (мини-игры).  
- **Механики:**  
  - **Поле зрения врагов:** Враги имеют поле зрения, которые реагируют на свет и шум. Если игрок попадает в поле зрения, начинается погоня.  
  - **Освещение:** Источники света (фонари, лампы) освещают окружающую территорию. Игрок может использовать тени для маскировки.  
  - **Документы:** Собирайте документы, чтобы завершить уровень. Каждый документ добавляет очки к вашему счёту.  

- **Очки и грейд:**  
  После завершения уровня вы получаете оценку (грейд) от `S` до `D`, которая зависит от количества собранных документов, оставшихся жизней и времени прохождения.  

### **Установка**  
1. **Клонируйте репозиторий:**  
  ```bash
  git clone https://github.com/feuille01/ShadowData_PyGame.git
  cd shadow-data
  ```
2. **Установите зависимости:**

  Проверьте, что у вас установлен Python 3.10+ и PyGame и выполните команду:
  ```bash
  pip install -r requirements.txt
  ```
3. **Запустите игру:**
  ```bash
  python main.py
  ```
### Как играть
1. **Стартовый экран:**  
   Нажмите "Начать игру", чтобы запустить первый уровень.
2. **Управление:**  
- `WASD`: Двигайтесь по карте.  
- Прячтесь за объектами, чтобы избежать обнаружения.
- Подбирайте документы, подходя к ним.  
3. **Цель:**  
Соберите все документы и доберитесь до точки выхода, избегая врагов и световых лучей.
4. **Экран результатов:**  
После завершения уровня вы увидите свой счёт и грейд (S-A-B-C-D).

### Технические требования  
- **Операционная система:** Windows.  
- **Python:** 3.10+.  
- **Библиотеки:** PyGame 2.x.
